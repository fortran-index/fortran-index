# Hackathon 4 (2025-12-15)

Building on feedback from the strategy session at Back to the Fortran Future II,
this hackathon will focus on making it clear on the
[fortran-lang](https://fortran-lang.org) website *why* someone would want to use
Fortran.

Number of attendees: 4
