# Hackathon 3 (2025-06-12)

> In this hackathon, we started to use GitHub issues to track what we were
> working on. See
> [#24](https://github.com/fortran-index/fortran-index/issues/24) for the
> tracker issue related to this hackathon.

In this hackathon, we focused on providing more information related to packages
and compilers. In particular, we wanted to make licensing and access
(open/closed-source) information more easily available.

Number of attendees: 4

- [#593](https://github.com/fortran-lang/webpage/pull/593): Provide more details
  on compilers.
