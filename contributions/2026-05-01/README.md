# Hack day at the Software Sustainability Institute's Collaborations Workshop (2026-05-01)

A hack day pitch was made to work on improving the
[package index](https://fortran-lang.org/packages/) at the
[SSI Collaborations Workshop](https://www.software.ac.uk/workshop/collaborations-workshop-2026-cw26).
While this wasn't selected as one of the hack day topics to work on, a group of
us focused on improvements for Modern Fortran training materials and also spent
some time improving the package index on the side.

Number of attendees: 5
